Comandos do sistema operacional

CD - (change dir) muda a pasta de direcionamento
echo - usado para adicionar um texto
> - é usado para criar o conteúdo do arquivo na tela
type - usado para exibir o conteúdo de um arquivo texto no console
dir - listar arquivos e pastas
/B - lista apenas o nome dos arquivos e pastas
/T:W - usado para especificar nome data e hora
